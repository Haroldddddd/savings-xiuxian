# 攒钱修仙 · savings-xiuxian

> 一个把省钱做成修仙升级游戏的 Claude Skill。
>
> Claude 全程扮演你的"师父"，陪你测灵根、立预算、验账记账。省下的每一分钱凝成灵石、炼化为修为，从练气一路修至飞升。

```
谁言节流非大道，积厘成山可叩神。
```

---

## 它解决什么问题

大多数记账工具都败在同一件事上——**枯燥**。省钱被包装成克制和牺牲，记账像给自己开罚单，坚持几天就放弃。

「攒钱修仙」换了个思路：**把每一次省钱，都变成一次修炼升级**。当省钱不再是"忍痛割舍"，而是"积攒修为、突破境界"，你会主动想记账。

## 世界观

你天生"灵窍闭塞"，无法像常人一样吸纳天地灵气修炼——唯独对「灵石」的吸纳之力远超常人。而你省下的每一分钱，都会凝成灵石。于是你拜入师门，走上一条以俭入道的逆天之路。Claude 扮演你的**师父**，用半文半白、引导式的口吻陪你修行。

## 核心玩法

- **🪄 测五行灵根，定省钱流派**：问卷测出你最容易花超的品类，作为"命定亲和战场"——该品类省下的钱修为 ×1.5。灵根一经测定不可更改。
- **💰 灵石坊市**：每次记账上传支付截图，如入坊市随机遇到一位灵石商人交易，把省下的钱凝成等额灵石、炼化为修为。截图是"交易信物"，无凭则修为不稳。
- **⚡ 十重境界与破境**：练气→筑基→金丹→元婴→化神→…→飞升。每次大境界突破都有专属破境场景与道诗，化神期起还要历天劫。
- **📈 道心系统**：连续记账养"道心"，决定灵石吸收效率（最高 ×1.2），断更则跌破 1。
- **🎲 随机奇遇**：记账后有概率触发限时省钱任务，限期内完成领额外灵石。
- **📚 内置省钱心法**：卡在某类开销或主动求教时，师父传授可立刻执行的省钱招法。

## 两条底线

- **数据诚实**：「实际攒下」永远是真实金额、1:1 不加任何倍率（诚信锚）；「修为」才是叠加灵根、道心加成的游戏进度。两者分开显示，加成让你升级更快，但绝不谎报你攒了更多钱。所有数值由脚本统一计算。
- **安全边界**：只读截图金额、不碰敏感信息；不提供个性化投资建议、不荐股——这是让省钱变有趣的游戏，不是理财顾问。

## 安装

将本仓库放入 Claude Code 的 skills 目录：

```bash
git clone https://github.com/Haroldddddd/savings-xiuxian.git ~/.claude/skills/savings-xiuxian
```

或克隆到任意位置后，按你的 Claude Code skills 加载方式引入。

## 使用

对 Claude 说「我想攒钱修仙 / 省钱」即可触发。师父会引导你：

1. **测灵根** —— 回答 3 道问卷，测出命定省钱战场。
2. **立预算** —— 报出每日各场景预算（早餐/午餐/通勤…），以及可选的月度预算（旅游/宠物/医疗…）。
3. **记账修炼** —— 报出实际花销 + 上传支付截图，省下的钱炼化为修为，突破境界。

数据默认存于 `~/.claude/savings-rpg/data.json`，可用环境变量 `SAVINGS_RPG_DIR` 覆盖。

底层引擎是单文件 Python（`scripts/savings.py`），负责全部状态与数值计算，无第三方依赖。

